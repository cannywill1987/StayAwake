# 国际化翻译 - 设置与主页面

## 本次范围
- 主导航：状态、会话、保持唤醒规则、设置。
- 页面标题与状态：就绪、空闲、开启、未开启、原生防睡眠断言状态。
- Status 页面：快速会话、睡眠行为、屏幕保护程序、电源来源、活动、菜单栏状态。
- Sessions 页面：会话启动器、会话历史、清除、空态。
- Stay Awake Rules 页面：电源规则、下载规则、App 规则、其他规则、按钮和动态状态。
- 运行中 App 选择弹窗：标题、搜索、空态、隐藏辅助进程、取消。
- 左下角电池提示。

## 涉及文件
- `lib/main.dart`
- `test/widget_test.dart`

## 实现方式
- 继续复用当前项目已有的轻量 `AppText` / `AppTextDelegate`。
- 没有引入 ARB 文件；当前项目尚未建立 `lib/l10n/*.arb` 源头。
- 使用 `AppText.pick(en, zh)` 覆盖英文和中文简体。

## 验证
- `rg -n "Text\\(\\s*'|label: const Text\\('|title: const Text\\('|hintText: '" lib/main.dart`
- `flutter analyze`
- `flutter test`
- `flutter build macos`

## 剩余风险
- macOS native 状态栏菜单 `AppDelegate.swift` 仍是中文硬编码，后续如果要支持英文菜单，需要给 native 侧补语言同步。
- 历史记录里已经保存过的旧英文日志不会自动迁移。
