## 修复记录

### UI-001 顶部空白
- 原因：主布局 `Row` 默认 `crossAxisAlignment.center`，右侧 `SingleChildScrollView` 按自身内容高度被垂直居中。
- 修改：主布局 `Row` 增加 `crossAxisAlignment: CrossAxisAlignment.stretch`，让右侧页面区域撑满高度并从顶部布局。
- after 截图：`docs/StayAwake/unknown-branch/06-UI测试/screenshots/status-after-260628.png`

### UI-002 国际化
- 修改：导航、页面标题、Status/Sessions/Rules 可见文案和运行中 App 弹窗接入 `AppText.pick(en, zh)`。
- 验证：中文环境下侧栏显示“状态 / 会话 / 保持唤醒规则 / 设置”，Status 页面主要文案显示中文。
